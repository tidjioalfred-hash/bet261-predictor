# BET 261 PREDICTOR 🇲🇬

Application de prédiction pour Bet 261 Madagascar — Aviator & Match Virtuel.

## 🚀 Déploiement sur Render.com (GRATUIT)

### Étape 1 — Créer un compte GitHub
Allez sur https://github.com et créez un compte gratuit.

### Étape 2 — Créer un repository
1. Cliquez sur **New repository**
2. Nom : `bet261-predictor`
3. Visibilité : **Public**
4. Cliquez **Create repository**

### Étape 3 — Uploader les fichiers
Uploadez ces 4 fichiers dans le repository :
- `app.py`
- `index.html`
- `requirements.txt`
- `render.yaml`

### Étape 4 — Déployer sur Render
1. Allez sur https://render.com
2. Créez un compte gratuit (avec votre compte GitHub)
3. Cliquez **New → Web Service**
4. Connectez votre repository `bet261-predictor`
5. Render détecte automatiquement la config
6. Cliquez **Deploy** ✅

### Résultat
Votre app sera live sur :
```
https://bet261-predictor.onrender.com
```
